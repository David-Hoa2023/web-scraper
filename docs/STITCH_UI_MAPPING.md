# Stitch UI Design Mapping

This document maps the 25 UI screens generated by Stitch (located in `images/stitch_web_scraper_all_tools_dashboard/`) to the existing and planned components of the Web Scraper Pro Chrome Extension.

## 1. Design System Alignment

| Property | Existing (Old) | Stitch (New) |
| :--- | :--- | :--- |
| **Primary Color** | `#00ff9d` (Neon Green) | `#13ec5b` (Bright Lime Green) |
| **Dark Background** | `#050505` | `#102216` (Deep Dark Green) |
| **Light Background** | `#ffffff` | `#f6f8f6` (Soft Grey-Green) |
| **Font Family** | `Segoe UI` | `Space Grotesk` |
| **Border Radius** | `12px` | `0.75rem` (12px) / `0.5rem` (8px) |
| **Aesthetic** | Dark Glassmorphism | Clean, Professional, Dark/Light High Contrast |

## 3. Implementation Priorities

1.  **Phase 1: Foundation (Current Tool)**
    -   Update `styles.css` with Stitch's color palette and `Space Grotesk` font.
    -   Refactor `popup.html` to include the new Sidebar and Header layouts.
2.  **Phase 2: Core Flow**
    -   Update `overlay.ts` to reflect the search and selection aesthetic from `data_extraction_selection_flow`.
3.  **Phase 3: Extended Features**
    -   Build out the "New Task" wizard in the popup using the 3 wizard designs.
    -   Implement the History and Webhooks tabs using their respective designs.
